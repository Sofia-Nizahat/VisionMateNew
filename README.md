# 🧠 About VisionMate  

VisionMate is an AI-powered assistive application designed to help visually impaired individuals navigate their surroundings safely and independently.

The system uses computer vision and voice interaction to detect obstacles, analyze the environment, and provide real-time audio feedback for safer mobility.

---

# 🌟 Key Features  

- 🎙 Voice-controlled interaction  
- 🧭 Real-time obstacle detection  
- 📍 Distance estimation alerts  
- 🔊 Instant audio guidance  
- ♿ Accessibility-first interface  
- 🔐 Secure user authentication

# 🚀 Getting Started  

## Prerequisites  

- Node.js 18+  
- npm or yarn  
- Camera & microphone access  

---

## Installation  

### 1️⃣ Clone the repository  

```bash
git clone https://github.com/your-username/visionmate.git
cd visionmate
```

### 2️⃣ Install dependencies  

```bash
npm install
```

### 3️⃣ Start the development server  

```bash
npm run dev
```

### 4️⃣ Open in browser  

```
http://localhost:5173
```

# 🔄 Application Workflow  

1. User activates voice command  
2. Camera captures real-time surroundings  
3. AI detects objects and obstacles  
4. Distance and risk analysis is performed  
5. Audio feedback guides the user
